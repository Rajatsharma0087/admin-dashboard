> Zero frameworks. Pure HTML, CSS & JavaScript.

[View Live Demo](https://rajatsharma0087.github.io/admin-dashboard/)

## ✨ Features
- 📊 Revenue chart (monthly/weekly toggle)
- 🍩 Traffic sources doughnut chart
- 📈 Animated stats counters
- 📋 Recent orders table with status badges
- 🏆 Top products with progress bars
- 🔔 Recent activity feed
- 🌙 Dark/Light mode with persistence
- 📱 Fully responsive + hamburger menu
- ⚡ Zero frameworks — pure HTML CSS JS

## 🛠️ Built With
| Technology | Purpose |
